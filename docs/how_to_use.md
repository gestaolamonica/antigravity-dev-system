# How to Use the Antigravity OS

1. **Bootstrap the System**: Clone this blank repository as a template for your new software project.
2. **Trigger the CTO**: Open your AI IDE (like Cursor or Gemini) referencing this workspace.
3. **Write the Prompt**: Send the initial message: `Antigravity: [Explain exactly what you want to build]`.
4. **Follow the Prompts**: The OS will dictate a PRD verification, Architectural review, and assign tasks directly to Lovable (Frontend), Claude (Backend), and Antigravity (Integration).
