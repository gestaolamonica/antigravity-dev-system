# Antigravity Dev System

This is the standard, templated organizational repository containing the **Antigravity AI CTO Operating System**.

## Contents
- `/antigravity_system/`: The core system configuration (`antigravity_os.md`, `project_template.md`, JSON schemas).
- `/docs/`: System architecture guidelines and structural rules.
- `/templates/`: New project templates.

## Initialization
To setup this core OS template for your team inside your GitHub Organization, you can run the bootstrap script:

```powershell
.\bootstrap.ps1 -OrganizationName "YourGitHubOrg" -Visibility "public"
```
